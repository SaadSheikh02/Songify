# Songify

Program that creates playlists from a collection of songs entered by the user.

# Usage

```bash
$ git clone https://github.com/SaadSheikh02/Songify
$ cd Songify/out
$ ./Songify
```

# Features
- Song Addition
- Song Removal
- Album Addition
- Album Removal
- Template Implementation
- Python Test Cases Implemented


# Compilation

```bash
$ make
$ cd ../out
$ ./Songify
```

## Dependencies

- g++
- make
